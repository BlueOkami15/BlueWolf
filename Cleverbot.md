#Chat
>
###Description:
**Chat** with the bot using the **Cleverbot API**.
###Aliases:
`clever, cleverbot, talk, @BotMention`
###Cooldown:  
`5s`
###Delete On Use:  
`False`
###Works in DM's:
`True`  
###Usage: 
`chat [text]`  
`@BotMention [text]`


#Unprefixed Direct Message
>
###Description:
**Chat** with the bot using the **Cleverbot API** in a **Direct Message**(Private message) **without** a prefix.
###Aliases:
`None`
###Cooldown:
`5s`  
###Delete On Use:  
`False`
###Works in DM's:  
`Only in DM's`
###Usage:  
`[text]`
